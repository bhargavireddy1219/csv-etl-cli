# CSV ETL CLI Tool

1. Created a GitHub repo with Financial transcations dataset
2. A simple Python CLI tool that reads a CSV dataset, cleans data(type casting, null handling) processes it, and writes output as dated Parquet files
3. Added logging, configuration via YAML and did 2 units with pytest
4. Created one feature branch, pull request to merge the feature branch with main branch
5. As I have used windows task scheduler is available to schedule daily run sample etl with Financial transcations dataset
6. Produced dated output in output folder
